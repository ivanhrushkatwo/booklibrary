Mini market book
====================

Run project
--------------------

1. Clone project.
```sh
$ git clone https://github.com/ivanhrushkatwo/booklibrary.git
```

2. Install dependencies.
```sh
$ cd booklibrary
$ pip install -r requirements.txt
```

3. Last step.
```sh
Configurate database(I use Postgresql)
$ python manage.py migrate
$ python manage.py runserver
```
